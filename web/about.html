Project Title: Secure Cloud Storage with Dual Custom S-Box Algorithm and HMAC Integrity Verification

This project proposes an advanced cryptographic solution for secure cloud storage by implementing a custom encryption algorithm known as the Dual Custom S-Box Algorithm. The focus is on ensuring the confidentiality, integrity, and authenticity of the stored data while maintaining high performance and minimal computational overhead.

Key Features:

  • Dual Custom S-Box Encryption Algorithm: A novel approach using custom S-boxes for higher security.
  • HMAC with SHA-256: Ensures data integrity and authentication.
  • SHA-256 for Key Derivation: Adds a layer of security for key generation.
  • Efficient Cloud Storage Encryption: Optimized processes for encryption and decryption.
  • User Authentication: Only authorized users can access encrypted data.
  • Forward and Backward Security: Protects data even if passphrases or keys are compromised.

Detailed Flow of the Project:

1. Data Encryption: The user’s passphrase is used to generate a secure encryption key. Data is encrypted using the Dual Custom S-Box Algorithm, and an HMAC is appended for integrity verification.

2. Data Decryption: Encrypted data is verified using the HMAC before being decrypted with the user’s passphrase.

3. Key Derivation: PBKDF2 with SHA-256 ensures unique and secure keys for each session.

4. HMAC Integrity Check: Ensures no tampering has occurred during transmission or storage.

Security Improvements and Performance Optimizations:

  • Enhanced Security: Resistant to cryptographic attacks like differential and linear cryptanalysis.
  • Optimized Performance: High efficiency with minimal computational impact.
  • Data Integrity: HMAC ensures data authenticity.

Conclusion:

This project presents a robust solution for secure cloud storage using advanced encryption and integrity verification techniques. The Dual Custom S-Box Algorithm and HMAC with SHA-256 offer strong security guarantees, making it an ideal system for sensitive cloud storage needs.
